Provides member directory, membership manangement and event management for small to mid-sized organizations.

== Description ==

[MemberFindMe](http://memberfind.me/)is a comprehensive website, membership and event management platform for small and mid-sized organizations 
such as chambers, professional groups, associations, and more.

This plugin creates shorttags for embedding the MemberFindMe member directory and event calendar on a WordPress site.

MemberFindMe subscription is required!

== Installation ==

1. Install the MemberFindMe plugin via the WordPress.org plugin directory or upload it to your plugins directory.
1. Activate the plugin
1. Under 'Settings' -> 'MemberFindMe', enter your MemberFindMe organization key, Stripe public key, Facebook API key

== Frequently Asked Questions ==

= Can I use this plugin without a MemberFindMe subscription? =

This plugin doesn't do anything without a MemberFindMe subscription. Please visit memberfindme.com for more information.
