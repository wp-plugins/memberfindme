Provides member directory, event calendar, membership manangement and event management for small to mid-sized organizations.

== Description ==

[MemberFindMe](http://memberfind.me/) provides a comprehensive website, membership management and event management solution for small to mid sized chambers, professional groups, associations and other member organizations. Create a smarter, more interactive website that features your members, promote your events, provide self-serve event registration, and organizes your membership information.

This plugin integrates MemberFindMe member directory and event calendar to your Wordpress site. MemberFindMe subscription is required.

**Member Directory**

* Smart directory search delivers more relevant search results
* Business card motif allows your members to express their identity
* Rich member profiles with
	- logos
	- pictures
	- in-depth information on products and services
	- contact information
	- location maps
	- links to social media
* Generate leads with user recommendations
* Create member deals or offers
* Share member profiles

**Event System**

* Infinitely scrolling calendar promotes browsing
* Color coded events on calendar
* One-page event registration increases likelihood of users completing registration
* Event registration payments using [Stripe](http://stripe.com)
* Automatic registration confirmation email with iCal calendar attachment (works with Outlook, gmail, etc)
* Create customized registration questions
* Group email event attendees
* Cancel and refund a single registration, or the entire event
* Download registration information to a spreadsheet
* Check-in attendees
* Registration and check-in will update to your member's timeline automatically

**Membership Management**

* Onboarding of member profiles saves you time and effort, and ensures all your members are well represented
* Labels to group your members by membership levels, committees and more...
* Member timeline allows you to see a member's history at a glance
* Membership billing, event registrations and event check-ins automatically update the member timeline
* Automated recurring membership billing using [Stripe](http://stripe.com)
* Membership data hosted separately on our secure servers with multiple redundancy and backup, for maximum safety and reliability

== Installation ==

1. Install the MemberFindMe plugin via the WordPress.org plugin directory or upload it to your plugins directory.
1. Activate the plugin
1. Under 'Settings' -> 'MemberFindMe', enter your MemberFindMe organization key, Stripe public key and Facebook API key

== Frequently Asked Questions ==

= Can I use this plugin without a MemberFindMe subscription? =

This plugin requires a MemberFindMe subscription. Please visit [MemberFindMe](http://memberfind.me/) for more information.

= Shortcodes =

* Member directory: \[memberfindme open="!directory"\]
* Event calendar: \[memberfindme open="!calendar"\]
* Event list: \[memberfindme open="!event-list"\]
* New member signup: \[memberfindme open="account/join"\]
* Account button: \[memberfindme button="account"\]
